# LinuxOnAndroid

* Run Linux on Android by Magisk & BusyBox.

## How to use?

* Install Magisk on your device.
* Install BusyBox by Magisk.
* Download this repository to "/data" directory.
* Execute "install.sh".
* Extract a linux rootfs to "/data/.linux". 
  * lxc image is recommended.
* Reboot your device.
